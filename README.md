# My Minimalist Dark Notepad
Este projeto nasceu da minha necessidade pessoal de encontrar um bloco de notas que equilibrasse funcionalidade e conforto visual.Sua interface consiste apenas em um campo para título e uma área de texto expansiva.
- Diferente da maioria dos editores online que podem saturar a visão com excesso de botões ou cores "cruas", este bloco de notas inicia nativamente em modo noturno para reduzir a fadiga ocular durante longas sessões de uso.
